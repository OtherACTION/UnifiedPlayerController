Detailed Description

Allows the player to push rigid bodies when colliding with them using a CharacterController. Applies a force to the rigid body based on the character's movement direction.

Attach this component to a GameObject with a CharacterController to enable pushing of rigidbodies on specified layers. Used by UnifiedPlayerController for interactive physics.
