Detailed Description

Allows zooming in and out of a third-person camera using the mouse scroll wheel. Uses the CinemachineThirdPersonFollow component to adjust the camera distance.

Attach this script to a GameObject with a CinemachineCamera. Assign the virtual camera in the inspector. Used in conjunction with UnifiedPlayerController for third-person gameplay.

See also
UnifiedPlayerController, CinemachineThirdPersonFollow
