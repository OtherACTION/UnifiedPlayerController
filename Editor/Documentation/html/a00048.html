Detailed Description

Handles player input for movement, looking, jumping, sprinting, and cursor state. Supports both Unity's legacy and new Input System.

This component should be attached to the player GameObject and is used by UnifiedPlayerController. It abstracts input handling so the controller can remain agnostic to the input system.
